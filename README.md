# Netflix

## Run Locally 

Clone the project  

~~~bash  
git clone https://github.com/sulekaptan/netflix.git
~~~

Go to the project directory  

~~~bash  
cd netflix
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the project  

~~~bash  
npm start
~~~
